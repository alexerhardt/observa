This is the backlog for the Observa project.

Every file is an issue.

The following format must be respected:
- Filename must be of the form: `OBS-<ID> <Human readable issue title>` 
- All files will have props:
	- id [string]
	- title [string]
	- status [picklist]
	- labels [multi-picklist]
- Ids must be sequential
- The file name may not contain Obsidian disallowed characters: `: [] # / \ !`
- Default state should be Backlog, unless otherwise specified
- There is no need to add labels, unless explicitly asked for

When creating an issue be sure to take the latest number from the filename and create the next sequential one.
